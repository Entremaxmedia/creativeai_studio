POST
/api/v1/jobs/createTask
Create Task
Create a new generation task

Request Parameters
The API accepts a JSON payload with the following structure:

Request Body Structure
{
  "model": "string",
  "callBackUrl": "string (optional)",
  "input": {
    // Input parameters based on form configuration
  }
}
Root Level Parameters
model
Required
string
The model name to use for generation

Example:

"gpt-image/1.5-image-to-image"
callBackUrl
Optional
string
Callback URL for task completion notifications. Optional parameter. If provided, the system will send POST requests to this URL when the task completes (success or failure). If not provided, no callback notifications will be sent.

Example:

"https://your-domain.com/api/callback"
Input Object Parameters
The input object contains the following parameters based on the form configuration:

input.input_urls
Required
array(URL)
Upload an image file to use as input for the API

Please provide the URL of the uploaded file; Accepted types: image/jpeg, image/png, image/webp; Max size: 10.0MB
Example:

["https://static.aiquickdraw.com/tools/example/1765962794374_GhtqB9oX.webp"]
input.prompt
Required
string
A text description of the image you want to generate

Max length: 3000 characters
Example:

"Change her clothing to an elegant blue evening gown. Preserve her face, identity, hairstyle, pose, body shape, background, lighting, and camera angle exactly as in the original image."
input.aspect_ratio
Required
string
Width-height ratio of the image, determining its visual form.

Available options:

1:1
-
1:1
2:3
-
2:3
3:2
-
3:2
Example:

"3:2"
input.quality
Required
string
Quality: medium=balanced, high=slow/detailed.

Available options:

medium
-
medium
high
-
high
Example:

"medium"
Request Example

cURL

JavaScript

Python
curl -X POST "https://api.kie.ai/api/v1/jobs/createTask" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-image/1.5-image-to-image",
    "callBackUrl": "https://your-domain.com/api/callback",
    "input": {
      "input_urls": [
        "https://static.aiquickdraw.com/tools/example/1765962794374_GhtqB9oX.webp"
      ],
      "prompt": "Change her clothing to an elegant blue evening gown. Preserve her face, identity, hairstyle, pose, body shape, background, lighting, and camera angle exactly as in the original image.",
      "aspect_ratio": "3:2",
      "quality": "medium"
    }
}'
Response Example
{
  "code": 200,
  "message": "success",
  "data": {
    "taskId": "task_12345678"
  }
}
Response Fields
code
Status code, 200 for success, others for failure
message
Response message, error description when failed
data.taskId
Task ID for querying task status
